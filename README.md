# AdventureWorks2019 SQL Project

---

## Project Overview

This project demonstrates practical SQL skills using Microsoft's **AdventureWorks2019** sample database.

The repository contains more than **150 SQL queries** covering SQL fundamentals, advanced analytical techniques, business reporting, and KPI development.

The primary goal of this project is to simulate real-world business analysis scenarios commonly encountered by Data Analysts and Business Intelligence Developers.
